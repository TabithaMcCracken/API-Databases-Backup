## Installation

Use the package manager pip to install requests, PyMySQL, and SQLAlchemy
```

pip install requests
pip install PyMySQL
pip install SQLAlchemy

```

Run DB_Setup.py to create a local database in MySQL 

```

python3 DB_Setup.py

```
